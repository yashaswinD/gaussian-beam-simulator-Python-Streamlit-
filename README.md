# Gaussian Beam Simulator

Interactive Python application for visualizing Gaussian beam
intensity profiles in transverse (x–y) and longitudinal (x–z) planes.

## Features
- Front and side Gaussian beam profiles
- Exact numerical beam parameters
- Unit-consistent SI calculations
- Research-oriented visualization

## Physics model
The beam is modeled under the paraxial approximation using
standard Gaussian optics.

pip install -r requirements.txt
